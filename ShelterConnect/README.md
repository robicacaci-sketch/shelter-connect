# Shelter Connect — Housing Readiness Assistant
## 2026 AI Hackathon | Challenge 1

### Team
Roberto Cacaci, Konnan Perez Roquel, Saarthank Uppal, Kevin Pinzoni

### Stack
- **Frontend**: React + TypeScript + Vite (port 5173)
- **Backend**: Node.js + Express + TypeScript + SQLite (port 4000)
- **AI**: Anthropic Claude API

### How to Run

**Backend:**
```
cd backend
npm install
echo "ANTHROPIC_API_KEY=your_key_here" > .env
npm run dev
```

**Frontend:**
```
cd frontend
npm install
npm run dev
```
Then open http://localhost:5173 and click "Continue with Demo Account".
